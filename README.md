# Couples Resort Redesign - Sihler Baldock
This repository showcases the redsesign of 3 mobile pages and 1 desktop page for the Couples Resort website. The redesign was part of the Final Integrated Project in Fanshawe College's Interactive Media Design Program.

## Installation
No installation required. Simply download the zip file and extract.

## Usage
Open index.html in the browser of your choice.
